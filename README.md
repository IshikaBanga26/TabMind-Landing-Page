# TabMind CLI — Landing Page

This repository contains the official landing page for **TabMind CLI** — a smart tab manager designed to help developers stop hoarding tabs and start finishing tasks.

---

## About TabMind CLI

TabMind CLI is a Python-based command-line tool that helps you:

* Save browser tabs with purpose
* Track why you opened them
* Set deadlines for tasks
* Get AI-powered reminders using GitHub Copilot

👉 Main Project: https://github.com/IshikaBanga26/TabMind
